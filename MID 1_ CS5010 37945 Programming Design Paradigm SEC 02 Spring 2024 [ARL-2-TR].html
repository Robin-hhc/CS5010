Implement the GeometricObject class and the Rectangle class according to the UML design below. 

 

Your Answer: 
public abstract class GeometricObject {

  private String color;

  private boolean filled;

  private java.util.Date dateCreated;

  public GeometricObject() {}

  public GeometricObject(color:String, filled:boolean) {
    this.color = color;
    this.filled = filled;
    this.dateCreated = new Date();
  }

  public String getColor(){ return this.color;}

  public void setColor(color: String) { this.color = color;}

  public boolean isFilled() { return this.filled;}

  public void setFilled(filled: boolean) { this.filled = filled;}

  public java.util.Date getDateCreated() { return this.dateCreated;}

  public String toString() {
    String ans;
    if (this.isFilled()) {
      ans = "GeometricObject is filled with color "+ this.color + ", created on "+ this.dateCreated;
    } else {
      ans = "GeometricObject is not filled with color "+ this.color + ", created on "+ this.dateCreated;
    }
    return ans;

}

public class Rectangle extends GeometricObject{

  private double width;

  private double height;

  public Rectangle() {
  }

  public Rectangle(width:double, height:double) {
    this.width = width;
    this.height = height;
  }

  public Rectangle(width:double, height:double, color:String, filled: boolean) {
    this.width = width;
    this.height = height;
    super.setColor(color);
    super.setFilled(filled);
  }

  public double getWidth() { return this.width;}
